# 多候选人相配度系统 (find-a-partner)

一个基于Web的多维度候选人相配度评估系统，帮助用户快速筛选和比较多个候选人与目标对象的匹配程度。

## 🚀 功能特性

- **多维度评估**：支持年龄、身高、性格、教育、兴趣、收入、居住城市和生活习惯等8个维度的匹配评估
- **自定义权重**：用户可以根据个人偏好调整各个评估维度的权重
- **硬条件惩罚机制**：可设置关键条件（如年龄差、身高下限、居住城市等），不符合条件的候选人将受到匹配度惩罚
- **实时计算**：所有更改（权重调整、候选人信息修改等）都会实时反映在匹配度结果中
- **可视化展示**：使用柱状图直观展示各维度对匹配度的贡献
- **响应式设计**：支持不同屏幕尺寸的设备访问
- **目标对象可编辑**：允许用户自定义目标对象的各项属性

## 🎯 如何使用

1. **打开系统**：直接在浏览器中打开 `index.html` 文件
2. **设置目标对象**：在"目标对象信息"部分填写或修改目标对象的各项属性
3. **调整权重**：使用滑块调整各个评估维度的权重（0-10）
4. **设置硬条件**：在"硬条件设置"部分配置关键条件和惩罚因子
5. **添加候选人**：点击"添加候选人"按钮，填写候选人信息
6. **查看匹配结果**：在仪表盘部分查看各候选人的匹配度排名和维度贡献图

## 📝 核心算法

### 匹配度计算

1. **各维度分数计算**：
   - 数值类型（年龄、身高、收入）：基于与目标值的差异计算
   - 文本类型（性格、教育、居住城市）：完全匹配得10分，否则得5分或0分
   - 兴趣/生活习惯：基于共同项的比例计算分数

2. **总匹配度**：
   ```
   总匹配度 = (Σ(维度分数 × 维度权重)) / (Σ(权重 × 10)) × 100%
   ```

3. **硬条件惩罚**：
   - 不符合硬条件的候选人，其总匹配度将乘以对应的惩罚因子
